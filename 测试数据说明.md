# 测试数据导入说明

## 测试数据准备

为了测试自动规划功能，您需要准备以下测试数据：

### 1. 计划模板数据 (plan_template)

**Excel表格字段说明：**

| 字段名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| name | String | 是 | 模板名称 | "Python学习计划" |
| total_time | Integer | 是 | 总天数 | 30 |
| remark | String | 否 | 备注 | "适合初学者" |
| is_enable | Boolean | 是 | 是否启用 | true |

**示例数据：**
```
name,total_time,remark,is_enable
Python学习计划,30,适合初学者,true
Java进阶计划,45,适合有基础的用户,true
前端开发计划,60,全栈开发,true
```

### 2. 任务集数据 (platform_task_group)

**Excel表格字段说明：**

| 字段名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| name | String | 是 | 任务集名称 | "基础语法" |
| description | String | 否 | 任务集描述 | "学习Python基础语法" |

**示例数据：**
```
name,description
基础语法,学习Python基础语法
面向对象,学习面向对象编程
Web开发,学习Django框架
数据分析,学习pandas和numpy
```

### 3. 任务数据 (platform_task)

**Excel表格字段说明：**

| 字段名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| name | String | 是 | 任务名称 | "变量和数据类型" |
| priority | Integer | 是 | 优先级(1-10) | 8 |
| background | String | 否 | 背景色 | "#FF5733" |
| suggested_time_start | String | 否 | 建议开始时间 | "09:00" |
| suggested_time_end | String | 否 | 建议结束时间 | "10:00" |
| remark | String | 否 | 备注 | "重点掌握" |
| annex_type | String | 否 | 附件类型 | "URL" |
| annex | String | 否 | 附件地址 | "https://example.com" |
| timing_type | String | 是 | 计时类型 | "POMODORO" |
| occupation_time | Integer | 否 | 预计耗时(分钟) | 60 |
| can_divisible | Boolean | 是 | 是否可分割 | true |

**计时类型说明：**
- `POMODORO`: 番茄钟类型
- `FREE_TIMING`: 自由计时类型  
- `UNTIMING`: 不计时类型

**示例数据：**
```
name,priority,background,suggested_time_start,suggested_time_end,remark,annex_type,annex,timing_type,occupation_time,can_divisible
变量和数据类型,8,#FF5733,09:00,10:00,重点掌握,URL,https://example.com,POMODORO,60,true
控制流程,7,#33FF57,10:00,11:00,理解条件判断,FILE,file.pdf,POMODORO,90,true
函数定义,9,#3357FF,14:00,15:30,核心概念,URL,https://example.com,FREE_TIMING,120,false
面向对象基础,6,#FF33F5,09:00,11:00,重要章节,IMG,image.jpg,POMODORO,120,true
```

### 6. 用户学习配置数据 (user_study_config)

**Excel表格字段说明：**

| 字段名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| user_id | Integer | 是 | 用户ID | 1 |
| plan_type | String | 是 | 规划类型 | "CUSTOM" |
| avg_hour_per_day | Float | 否 | 平均每日小时数 | 4.0 |
| auto_plan_mode | String | 是 | 自动规划模式 | "MODE1" |
| max_daily_minutes | Integer | 是 | 每日最大学习时间(分钟) | 480 |
| is_locked | Boolean | 是 | 是否锁定 | false |

**规划类型说明：**
- `CUSTOM`: 自定义时间类型
- `AVG_HOUR`: 平均每日时长类型

**自动规划模式说明：**
- `MODE1`: 任务集 + 优先度
- `MODE2`: 任务集 + 顺序
- `MODE3`: 不考虑任务集 + 优先度
- `MODE4`: 不考虑任务集 + 顺序
- `MODE5`: 全局优先度

**示例数据：**
```
user_id,plan_type,avg_hour_per_day,auto_plan_mode,max_daily_minutes,is_locked
1,CUSTOM,,MODE1,480,false
2,AVG_HOUR,4.0,MODE2,360,false
3,CUSTOM,,MODE3,600,true
```

## 测试用例示例

### 用例1：Python学习计划

**模板信息：**
- 名称：Python学习计划
- 总天数：30天
- 任务集：基础语法、面向对象、Web开发

**任务分布：**
- 基础语法：10个任务，每个30-60分钟
- 面向对象：8个任务，每个60-90分钟
- Web开发：12个任务，每个45-120分钟

**用户配置：**
- 每日最大学习时间：480分钟
- 规划模式：MODE1

**预期结果：**
- 任务按任务集和优先级排序
- 每日学习时间不超过8小时
- 30天内完成所有任务

### 用例2：高强度学习计划

**模板信息：**
- 名称：高强度学习计划
- 总天数：15天
- 任务集：快速入门、深度实践

**任务分布：**
- 快速入门：20个任务，每个15-30分钟
- 深度实践：10个任务，每个120-180分钟

**用户配置：**
- 每日最大学习时间：600分钟
- 规划模式：MODE3

**预期结果：**
- 任务按全局优先级排序
- 每日学习时间不超过10小时
- 15天内完成所有任务
